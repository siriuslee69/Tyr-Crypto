What this shell is for

One shell, many custom-crypto notes

This root page loads standalone algorithm documents from the corresponding subfolders under docs/. The idea is that every algorithm can keep its own detailed writeup, while the shell provides the shared left-menu docs layout, typography, and LaTeX rendering surface.

Doc Shell Rules

  • Start high-level, then descend into modules, then into hot loops.
  • Separate speed work, security work, and rollback decisions explicitly.
  • Keep numeric examples small enough to recompute by hand.
  • Use the left-menu search and state filters to move between writeups.
